# 🌍 Srishti Travels

A modern and responsive travel website built using **HTML, CSS, and JavaScript**. The website provides an attractive user interface for exploring travel destinations, tour packages, and travel services.

## ✨ Live Demo

🔗 https://srishtitravallss.netlify.app

## 📌 Features

- 🏠 Responsive Home Page
- 🌍 Popular Travel Destinations
- 🏖️ Tour Packages Section
- 📸 Beautiful Image Gallery
- 🔍 Search Destination
- 📱 Mobile-Friendly Design
- 🎨 Modern UI/UX
- 💫 Smooth Animations
- 🖱️ Interactive Navigation
- 📞 Contact Section
- ⭐ Customer Reviews
- 🗺️ Travel Information
- 📅 Easy Navigation
- ⚡ Fast Loading Website

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/srishtisonawane2007/yatra-application.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
